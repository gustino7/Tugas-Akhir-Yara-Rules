---------------

Wide: 
Definisi    : String dalam format wide adalah string yang menggunakan encoding 16-bit (2 byte per karakter), seperti UTF-16. Ini sering digunakan dalam file Windows untuk menyimpan string Unicode.
Penggunaan  : Digunakan untuk mencocokkan string yang disimpan dalam format wide (misalnya, string Unicode dalam file PE).
